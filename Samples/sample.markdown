# Release Notes

This `.markdown` file mirrors `.md` support and uses the same renderer.

## Added

- Headings
- Paragraphs
- Bullet lists
- Blockquotes
- Fenced code blocks
- Inline **bold**, *italic*, `code`, and links

> Useful for checking that Finder routes `.markdown` files to QuickLookExtended.

## Example

```yaml
name: QuickLookExtended
formats:
  - md
  - markdown
rendered: true
```

See [Markdown Guide](https://www.markdownguide.org/) for the full format.
