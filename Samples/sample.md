# Markdown Preview Demo

This file checks the rendered Markdown preview in QuickLookExtended.

## Inline Formatting

Text can be **bold**, *italic*, or use `inline code`.

Links render as clickable-looking text: [QuickLookExtended](https://github.com/).

> Blockquotes get a left border and softer text color.

## Lists

- Markdown is rendered instead of shown as raw markup.
- Text selection and copy still work in Quick Look.
- Large files can still fall back to plain text for speed.

## Code Block

```swift
let file = "sample.md"
let preview = "rendered markdown"
print("\(file): \(preview)")
```
